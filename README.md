# Q0 Calcuator

<img src="https://github.com/skytomo221/Q0-Calculator/blob/master/images/icon.png?raw=true" width="200px">

**Q0 Calcuator** は Java の課題のために作られた電卓プログラムです。  
そのため、単位が確定するまでは、このソースコードの利用をお控えください。

## 現在対応済みの型

|型名|説明|
|:-:|:-:|
|`Integer`|整数型|
|`Double`|浮動小数点型|
